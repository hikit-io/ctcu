# ctcu
Copy title and url to json on webpage for safari extension. 

# Usage

1. Click extension button

<img width="707" alt="image" src="https://user-images.githubusercontent.com/42961884/207051967-0bed9cd1-76cb-4634-8a61-631578a5ae5f.png">

2. Paste anywhere

```json
{
  "title" : "Google",
  "link" : "https://www.google.com/?client=safari"
}
```
